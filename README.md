## Overview

Bunkbuddy is a cross-platform mobile application built using React Native (Expo) that helps students efficiently track attendance, calculate percentages, and plan allowable absences based on academic requirements.

The application is designed to simplify academic planning through real-time insights and an intuitive interface.

---

## Features

- Real-time attendance tracking and percentage calculation  
- Smart bunk planning based on configurable thresholds (75%, 85%, 95%)  
- Calendar-based date range selection for planning  
- Weekly and overall attendance insights  
- Clean and structured card-based user interface  
- Firebase integration for scalable backend support
